# Закрывающий тег

Учебный проект Яндекс Практикума — интерактивная страница-дневник, посвящённая пути обучения веб-разработке. Каждая карточка отражает этап учёбы: от нулевого спринта до текущего момента.

## Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (CSS-переменные, `clamp()`, логические свойства, `@supports`, CSS-фильтры, CSS-анимации)
- JavaScript (интерактивные кнопки Like, модальное окно `<dialog>`)
- Адаптивная вёрстка (медиавыражения)
- Шрифты: Inter Variable, Press Start 2P

## Структура проекта

```
├── index.html            — основная страница
├── styles/
│   ├── style.css         — основные стили
│   ├── globals.css       — глобальные стили
│   ├── variables.css     — CSS-переменные и темы
│   └── animations.css    — анимации
├── scripts/
│   └── like.js           — логика кнопки Like
├── fonts/                — шрифты проекта
├── images/               — изображения
└── svg/                  — SVG-иконки
```
